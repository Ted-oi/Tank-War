# 坦克大战 - Tank Battle

经典坦克大战游戏的 HTML/JavaScript 实现。

## 游戏特点

- 🎮 经典俯视角坦克对战
- 🤖 AI 敌方坦克，逐步提升难度
- 🧱 可破坏的砖墙、坚不可摧的钢墙
- 🌊 水域障碍物
- 🌿 提供视觉掩护的草丛
- 🦅 需要保护的我军基地
- 💥 爆炸和粒子特效
- 🔊 音效反馈
- 📈 计分系统和多关卡

## 操作方式

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → | 移动坦克 |
| 空格键 | 射击 |
| P | 暂停/继续 |
| Enter | 开始/重新开始 |

## 关卡

- **第1关**: 入门级，砖墙为主
- **第2关**: 引入钢墙和水域
- **第3关**: 复杂迷宫布局

## 如何游玩

直接在浏览器中打开 `index.html` 即可游玩，无需任何依赖或服务器。
